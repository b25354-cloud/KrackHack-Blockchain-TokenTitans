# PayStream V2 - Corporate Payroll Streaming Protocol

PayStream V2 is a real-time payroll streaming protocol built on the HeLa Testnet. It allows organizations to stream salaries to employees second-by-second, automate tax collection, issue instant or time-locked bonuses, and enables employees to withdraw funds or request off-ramp conversions to fiat currencies.
